# Global Islamic Pilgrimage Dataset Analysis

## Overview

This project explores global Islamic pilgrimage trends using a dataset containing information about pilgrim demographics, travel costs, participation, satisfaction levels, and economic impact.

The goal of this analysis is to identify patterns in pilgrimage participation and understand the factors influencing pilgrim experiences across different countries and years.

## Dataset

The dataset contains information such as:

- Country
- Year
- Pilgrimage Type
- Number of Pilgrims
- Male/Female Participation
- Average Age
- Travel Cost
- Satisfaction Score
- Economic Impact

## Objectives

- Analyze pilgrimage participation trends over time
- Compare participation across countries
- Study demographic distributions
- Investigate travel costs and satisfaction
- Examine the economic impact of pilgrimages
- Identify notable patterns and relationships in the data

## Key Findings

- Participation varies significantly between countries.
- Economic impact generally increases with pilgrim volume.
- Travel costs differ considerably across regions.
- Demographic patterns remain relatively stable over time.
